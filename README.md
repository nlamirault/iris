# Iris

[![Circle CI](https://circleci.com/gh/nlamirault/iris.svg?style=svg)](https://circleci.com/gh/nlamirault/iris)

Iris is a [cadvisor][] client to monitor [Docker][] containers.

## Usage

Launch the web service :

	$ iris -d


## Development

* Initialize environment

        $ make init

* Build tool :

        $ make build

* Launch unit tests :

        $ make test
